# یادداشت نسخه بازطراحی تسک و مالی

این نسخه شامل بازطراحی گردش کار تسک، ثبت و تأیید ساعت، نمودارهای ساعت اعضا و پروژه‌ها و دفتر درآمد و هزینه است.

## تغییرات اصلی

- تفکیک تسک‌های «ساخته‌شده توسط من»، «محول‌شده به من» و «در انتظار تأیید من»
- نمایش محول‌کننده، مسئول انجام، ارزش تسک و ساعت تأییدشده/در انتظار تأیید
- سه وضعیت کاری «شروع‌نشده»، «در حال انجام» و «انجام‌شده» و آرشیو خودکار پس از تأیید مدیر
- امکان ثبت ساعت توسط سازنده یا مسئول تسک و لحاظ‌شدن ساعت فقط پس از تأیید مدیر
- دریافت زمان گزارش به واحد ساعت (با پشتیبانی از مقدار اعشاری) و نمایش فوری کل زمان ثبت‌شده روی تسک
- پنل فعال ثبت و تاریخچه ساعت برای هر تسک و امکان ویرایش تسک توسط سازنده، مسئول یا مدیر
- نمایش دفتر درآمد و هزینه برای مدیر سازمان و مدیر پروژه
- نمودار ساعت اعضای دپارتمان و پروژه‌ها در بازه زمانی با نوار درصدی
- دفتر اسناد درآمد و هزینه با گروه‌بندی، فیلتر، جمع‌بندی و اتصال اختیاری سند به پروژه
- migration دیتابیس برای ارزش تسک و جداول مالی

## راه‌اندازی

راه‌اندازی پروژه مانند نسخه اصلی است. پیش از اجرای نسخه جدید، migrationهای Alembic را اعمال کنید:

```bash
cd backend
alembic upgrade head
```

جزئیات فنی بیشتر در `docs/TASKS_AND_FINANCE_REDESIGN.md` قرار دارد.

## کنترل کیفیت انجام‌شده

- build تولیدی فرانت‌اند موفق
- ۵۵ تست فرانت‌اند موفق
- بارگذاری کامل برنامه بک‌اند و ثبت ۸۳ مسیر API موفق
- جمع‌آوری ۱۲۷ تست بک‌اند موفق
- تأیید یکپارچگی migrationها با یک head

اجرای کامل تست‌های بک‌اند به سرویس PostgreSQL نیاز دارد.
